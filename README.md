# Nummer — Pro Forma Builder

Build and compare 5-year pro forma financial statements from your assumptions.

## Open the app

**Local file:** open `index.html` in your browser (double-click or drag into Chrome/Safari).

**Live site:** [https://quicktest007.github.io/nummer/](https://quicktest007.github.io/nummer/)

If you still see an old “Editor / Balance Sheet” UI, hard-refresh: **Cmd+Shift+R** (Mac) or **Ctrl+Shift+R** (Windows).

## How it works

1. **Pro Forma** (default) — 5-year income statement, cash flow, and balance sheet
2. **Assumptions** — revenue growth, margins, balance sheet, and working capital inputs
3. **Analytics** — charts for revenue, margins, cash flow, and ratios
4. **Compare** — Base, Upside, and Downside scenarios side by side

Use the **Scenario** bar (Base / Upside / Downside) to stress-test results. Click **Recalculate** after changing assumptions.

## Export

**Export PDF** opens a print-ready pro forma package (income statement, cash flow, notes). Choose “Save as PDF” in the print dialog.
